ASL Detector

What is ASL Detection?

ASL (American Sign Language) Detection uses advanced machine learning with MediaPipe hand tracking and a trained neural network model to recognize hand gestures in real-time from your webcam.

Technology Stack

FastAPI: High-performance web server
OpenCV: Real-time computer vision
MediaPipe: Hand landmark detection
TensorFlow/Sklearn: ML model

How to Use:

  1. Position your hand clearly in front of your webcam
  2. Ensure good lighting for accurate detection
  3. The system will detect your hand and display a bounding box
  4. The predicted ASL character will appear above the bounding box
  5. You can use both hands simultaneously (up to 2 hands)
  6. Special gestures like 'space' and 'delete' are also supported
